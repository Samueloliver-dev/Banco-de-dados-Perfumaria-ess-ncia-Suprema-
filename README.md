# Banco-de-dados-Perfumaria-ess-ncia-Suprema-
Perfumaria Essência Suprema – Banco de Dados SQL  Projeto de SQL prático que simula o gerenciamento de uma perfumaria. Inclui criação de tabelas, relacionamentos, inserção de dados, consultas avançadas e manipulação com UPDATE/DELETE. Ideal para treinar bancos de dados relacionais e comandos DML/DDL em um cenário realista.
